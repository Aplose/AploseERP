# Documentation utilisateur

Bienvenue dans la documentation d'AploseERP. Cette section vous aide à prendre en main l'application et à utiliser les principaux écrans.

## Contenu

- **Prise en main** : premiers pas après création de compte
- **CRM & Tiers** : gestion des tiers et contacts
- **Commerce** : devis, commandes et factures
- **Catalogue** : produits et catégories
- **Projets & Agenda** : projets, tâches et calendrier

Utilisez le menu à gauche pour naviguer entre les chapitres.
