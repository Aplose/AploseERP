# CRM & Tiers

Le module CRM permet de gérer les **tiers** (clients, fournisseurs) et les **contacts** associés.

## Tiers

Un tier représente une entreprise ou une personne avec laquelle vous travaillez. Vous pouvez indiquer :

- Raison sociale, code, forme juridique
- Adresse, email, téléphone
- Numéro SIRET ou identifiant (optionnel)

Les tiers sont utilisés comme clients sur les devis et factures, ou comme fournisseurs sur les commandes d'achat.

## Contacts

Les contacts sont liés à un tier. Ils permettent d'associer plusieurs personnes (nom, prénom, email, téléphone) à une même société. Lors de la création d'un devis ou d'une commande, vous choisissez le tier et éventuellement le contact concerné.

## Bonnes pratiques

- Saisir l'adresse complète du tier pour les en-têtes de documents
- Utiliser les champs complémentaires (si activés) pour des informations métier spécifiques
