# Documentation intégrateur

Cette section s’adresse aux intégrateurs qui souhaitent connecter des systèmes externes à AploseERP (API, SSO, flux d’inscription, etc.).

## Contenu

- **API REST** : présentation des APIs, authentification et exemples d’appels

## Bonnes pratiques

- Utiliser l’authentification prévue (token, session) et ne pas exposer d’identifiants
- Respecter les limites de débit si documentées
- Consulter la documentation développeur pour l’architecture technique
