# Commerce

Le module Commerce regroupe les **devis**, **commandes** et **factures**.

## Devis

Un devis est un document proposé au client avec des lignes (produits ou services), quantités et prix. Vous pouvez :

- Créer un devis à partir d’un tier (client) et d’un contact
- Ajouter des lignes depuis le catalogue ou en saisie libre
- Marquer le devis comme « envoyé » puis « accepté » ou « refusé »
- Convertir un devis accepté en commande

## Commandes

Une commande peut être créée manuellement ou à partir d’un devis. Elle contient les lignes commandées, les dates et les conditions. Les commandes permettent de suivre les livraisons et de générer des factures.

## Factures

Les factures sont des documents de vente ou d’achat. Vous pouvez :

- Créer une facture de vente (client) ou d’achat (fournisseur)
- Lier une facture à une commande
- Enregistrer les paiements (montant, date, mode) pour suivre le solde restant

Les statuts (brouillon, envoyé, validé, payé, etc.) reflètent le cycle de vie du document.
