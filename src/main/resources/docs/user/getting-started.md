# Prise en main

Après la création de votre compte, connectez-vous avec l'email et le mot de passe de l'administrateur.

## Tableau de bord

Le tableau de bord affiche un résumé de l'activité : vous y accédez après connexion. Il constitue le point d'entrée principal.

## Menu principal

La barre de navigation donne accès aux modules :

- **CRM** : Tiers, contacts
- **Catalogue** : Produits, catégories, devises
- **Commerce** : Devis, commandes, factures
- **Projets** : Projets et tâches
- **Agenda** : Calendrier et événements
- **Administration** : Utilisateurs, rôles, paramètres, dictionnaires (selon vos droits)

## Paramétrage de base

Avant de saisir des devis ou factures, il est recommandé de :

1. Compléter les **paramètres** du tenant (raison sociale, adresse, langue)
2. Vérifier les **devises** et **modes de paiement** dans les dictionnaires
3. Créer au moins un **tier** (client ou fournisseur) et un **contact**

## FAQ courte

**Comment ajouter un utilisateur ?**  
Allez dans Administration → Utilisateurs → Créer. Réservé aux administrateurs.

**Comment changer la langue ?**  
Profil (icône utilisateur) ou paramètres du tenant.

**Où sont les modèles d'email ?**  
Administration → Modèles d'email (bienvenue, onboarding).
