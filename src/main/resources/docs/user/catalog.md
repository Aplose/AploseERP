# Catalogue

Le catalogue gère les **produits** (et services) et les **catégories**.

## Catégories

Les catégories permettent d’organiser les produits (ex. : Équipement, Prestations, Consommables). Vous pouvez créer une arborescence et associer chaque produit à une catégorie.

## Produits

Pour chaque produit vous pouvez définir :

- Code, nom, type (produit, service, consommable)
- Catégorie, unité de mesure
- Prix de vente et éventuellement prix d’achat
- Taux de TVA, code-barres
- Options : vendable, achetable, suivi de stock avec seuil d’alerte

Les produits sont ensuite sélectionnables dans les lignes de devis, commandes et factures.

## Devises

Le module permet de gérer les devises (code, symbole, décimales). La devise par défaut du tenant est utilisée sur les documents si aucune autre n’est choisie.
